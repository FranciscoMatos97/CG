Instalação do projeto:
Através do CLion correr o projeto "generator" indicando a figura que deseja desenhar, os seus argumentos e o ficheiro ".3d" onde pretende guardar os vertices.

De seguida, correr o projeto "engine" indicando o ficheiro "scene.xml" que contém os figuras a desenhar.
